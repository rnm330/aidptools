# 🐑 小羊 AIDP 做题工具

Chrome 浏览器扩展，让 AIDP 做题轻松一点。

## ✨ 功能一览

**📊 自动计数** — 页面上点「提交」自动 +1，「上一题」自动 -1（禁用/倒计时状态不触发），实时显示进度、完成率和节奏提示

**🏷️ 快捷标签** — 把常用标签保存下来，一键自动填充，告别重复手选

**🐟 摸鱼模式 v2** — 手动/自动摸鱼双模式，120秒刷新 + 静音循环播放视频 + 10秒循环模拟点击选项卡保持活跃，摸鱼自身刷新后自动恢复

**⏱️ 自动摸鱼（双开关）** — 两个独立开关分别控制：30秒无操作自动摸鱼 / 切换后台自动摸鱼，可单独或同时开启

**📦 押后提交** — 押后列表页面自动按设定间隔逐题提交，支持倒计时显示、视频静音循环、完成后统计汇总

**⚙️ 自定义设置** — 可修改每日目标和工作时间段，输入任务 ID 快速跳转

## 🏷️ 快捷标签怎么用

**保存标签：** 在页面上手动选好标签 → 点击「快捷选择」→ 自动保存，浮动窗口随即出现

**新增标签：** 页面上选好新标签 → 点击「快捷选择」→ 选择「新增标签」，自动追加并去重

**替换标签：** 页面上选好标签 → 点击「快捷选择」→ 选择「替换标签」，覆盖之前保存的

**清空标签：** 页面上不要选标签 → 点击「快捷选择」→ 选择「清空标签」

**一键填充：** 浮动窗口中直接点击标签按钮，自动完成级联选择。支持连续点击自动排队、已存在自动跳过、高频标签紫色高亮。窗口可拖拽、缩放、最小化，位置跨天保持稳定。

## 🐟 摸鱼模式怎么用

### 手动摸鱼
点击底部工具栏的「摸鱼」按钮即可开启。开启后会：
- 每 **120 秒**刷新页面（摸鱼自身的刷新会自动恢复摸鱼状态）
- **静音 + 循环播放**视频
- 每 **10 秒**模拟点击题目区域下拉框输入框保持页面活跃
- 停止后恢复视频原始静音、播放和循环状态

### 自动摸鱼（两个独立开关）
设置面板中可分别或同时开启：

| 开关 | 触发条件 |
|------|---------|
| **空闲30秒自动摸鱼** | 30秒无鼠标/键盘/滚轮操作 |
| **切换后台自动摸鱼** | 切换标签页 / 最小化窗口 |

- 开启押后提交功能期间，两个自动摸鱼开关**暂时失效**（不影响开关本身的状态）
- 摸鱼模式开启后**不会因为用户操作而自动退出**，必须手动点击「停止」按钮

### 注意事项 ⚠️
- **不要关闭当前标签页** — 关闭后摸鱼停止
- **不要提交题目** — 提交会计入做题计数
- **不要打开左侧侧边栏** — 可能导致页面行为异常
- **不要手动刷新页面（F5）** — 手动刷新会停止摸鱼（只有摸鱼自己的120s倒计时刷新才会保留状态）

### 多窗口独立
每个浏览器标签页的摸鱼状态独立存储（sessionStorage），互不影响。

## 📦 押后提交怎么用（v0.5 新增）

### 使用前提
1. 当前页面必须是**押后列表页面**（URL 包含 `/freeze-list/` 且页面有「连续领题」「提交」按钮）
2. 非押后页面点击「押后提交」会提示：`请打开押后列表处理题目！`

### 操作步骤
1. 打开押后列表页面
2. 点击工具栏的 **「押后提交」** 按钮 → 弹出设置面板
3. 设置提交间隔（秒），默认 **160 秒**
4. 点击 **「开始」** 启动自动提交

### 运行中的行为
启动后会依次执行：

| 步骤 | 说明 |
|------|------|
| ① 暂停当前窗口摸鱼 | 如果摸鱼在运行则停止（开关不受影响） |
| ② 自动摸鱼暂时失效 | 空闲/后台自动摸鱼两个开关暂停生效 |
| ③ 打开第一题弹窗 | 点击表格首行的「处理」，弹出做题弹窗 |
| ④ 倒计时等待 | 右上角实时显示倒计时 `XXs 后提交` |
| ⑤ 点击「提交」 | 在弹窗内点击「提交」（非「干预提交」） |
| ⑥ 弹窗自动切题 | 提交成功后，弹窗自动加载下一题（利用连续领题机制） |
| ⑦ 循环 ④→⑤→⑥ | 直接在弹窗内继续点「提交」，不再重复开弹窗 |
| ⑧ 全部完成 | 表格为空时自动停止，弹出完成汇总 |

### 右上角运行提示框
```
📦 158s 后提交
已提交：3 题 | 已用：8分钟 | 视频已静音循环
```
- 实时更新倒计时、已提交数、已用时间

### 完成后的保留提示框
```
🎉 押后完成！
✅ 押后提交完成！共 5 题，耗时 12.3 分钟
[关闭]
```
- 黄色主题，不自动消失，手动点「关闭」才消失

### 与摸鱼的联动关系

| 场景 | 摸鱼行为 | 自动摸鱼开关 |
|------|---------|-------------|
| 开启押后 | 暂停当前窗口摸鱼 | 暂时失效 |
| 押后运行中 | 仅静音+循环播放视频（不模拟点击、不刷新） | 不触发 |
| 停止押后 | 统一关闭（不恢复摸鱼） | 立即恢复正常 |
| 其他窗口 | 完全不受影响 | 正常生效 |

### 间隔设置说明
- 默认 **160 秒**，可自由输入任意正整数
- < 60 秒 → 黄色提示"提交时间太短，可能导致单题处理时间过短！"
- > 160 秒 → 黄色提示"提交时间太长，可能导致人效不达标！"
- 不阻止任何值，点「开始」即生效，无需手动保存

## 📦 安装

1. 下载 [Releases](https://github.com/rnm330/aidptools/releases) 中的 zip 并解压
2. 打开 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择解压后的文件夹
5. 上传到 Chrome 商店时需要填写隐私权声明（见下方）

### Chrome 商店隐私权说明

**host_permissions：** 仅请求访问 `https://aidp.bytedance.com/*`，用于在 AIDP 平台注入内容脚本、读取 DOM 元素和注入自定义 UI。不会收集用户数据或向外部发送请求。

**storage：** 用于在本地保存配置和状态数据（每日目标、工作时间、已完成计数、快捷标签模板、窗口位置等）。所有数据仅存于本地浏览器，不涉及网络传输。

- 其他浏览器窗口完全不受影响

**🐟 摸鱼模式优化：**
- 自动摸鱼拆分为**两个独立开关**：空闲30秒 / 切换后台，可单独或同时开启

- 停止摸鱼时**完整恢复视频状态**：静音、播放、循环全部还原
