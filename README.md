# 小羊 AIDP 做题工具

Chrome 浏览器扩展，为 AIDP 平台做题提供进度追踪和快捷标签填充功能。

## 功能

### 进度追踪
- **实时计数** — 自动检测「提交」+1、「上一题」-1（禁用状态不计数）
- **智能计算** — 根据工作时间自动计算应完成数量
- **手动编辑** — 点击完成数可直接修改
- **可视化进度条** — 预期进度（黄色）+ 完成进度（绿色）

### 快捷标签填充
- **一键保存** — 提取页面上已选标签，保存为快捷填充模板
- **快速填充** — 点击浮动窗口中的标签按钮，自动完成级联选择操作
- **智能队列** — 支持连续快速点击，自动排队执行
- **前置检查** — 标签已存在则跳过，不重复执行
- **自动重试** — 被操作中断后自动重试（最多5次），确保填充成功
- **浮动窗口** — 独立浮动面板，支持拖拽移动、自由缩放、最小化
- **合并标签** — 支持替换或新增标签模式，智能去重
- **使用统计** — 高频使用的标签自动高亮显示（动态阈值）

### 交互设计
- **现代暗色主题** — 深色渐变背景，精致按钮质感
- **点击反馈** — 按钮悬停光环 + 按压缩放，触感清晰
- **气泡提示** — 操作结果以底部气泡形式展示

## 安装

1. 下载本仓库
2. 打开 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目文件夹

## 使用

- 底部工具栏实时显示做题进度
- 点击「设置」修改每日目标和工作时间
- 输入任务 ID 可快速跳转
- 点击「快捷选择」保存页面上已选的标签
- 保存后自动生成浮动窗口，点击标签按钮快速填充

## 默认配置

- 每日目标：160 题
- 工作时间：10:00-12:00，13:00-19:00
