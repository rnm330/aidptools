# 小羊 AIDP 做题工具

Chrome 浏览器扩展，为 AIDP 平台做题提供进度追踪、快捷标签填充和摸鱼模式。

## 能干什么

- **自动计数** — 页面上点「提交」自动 +1，实时显示进度和完成率
- **快捷标签** — 把常用标签保存下来，一键自动填充，不用每次手选
- **摸鱼模式** — 开启后每180秒自动刷新页面，刷新后自动恢复

## 安装

1. 下载 Releases 中的 zip 并解压
2. 打开 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择解压后的文件夹

## 快捷标签使用指南

### 保存标签

1. 在 AIDP 做题页面上，手动选择好需要的标签
2. 点击底部工具栏的「快捷选择」按钮
3. 如果是第一次使用，会直接保存当前页面上已选的所有标签
4. 保存后会自动生成浮动窗口，点击标签按钮即可一键填充

### 新增标签

1. 在页面上选好新的标签
2. 点击「快捷选择」
3. 选择「新增标签」— 新标签会追加到已保存的列表中（自动去重）

### 替换标签

1. 在页面上选好标签
2. 点击「快捷选择」
3. 选择「替换标签」— 用当前页面上的标签替换掉之前保存的

### 清空标签

1. 确保页面上没有选中任何标签
2. 点击「快捷选择」
3. 选择「清空标签」

或者页面有标签时，点击「快捷选择」后选择「替换标签」也可以清空。

### 标签填充

- 浮动窗口显示在页面上，点击标签按钮会自动完成级联选择
- 支持连续快速点击，自动排队执行
- 标签已存在会自动跳过，不会重复
- 浮动窗口可以拖拽移动、缩放、最小化
- 高频使用的标签会自动高亮为紫色

## 其他功能

- **设置** — 点击「设置」可修改每日目标和工作时间段
- **任务跳转** — 输入任务 ID，点击「跳转」快速打开对应任务
- **摸鱼模式** — 点击「摸鱼」开启180秒自动刷新，再次点击停止；刷新后自动恢复，每个窗口独立

## 默认配置

- 每日目标：160 题
- 工作时间：10:00-12:00，13:00-19:00
