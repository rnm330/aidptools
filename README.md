# 小羊 AIDP 做题工具

Chrome 浏览器扩展，帮助在 AIDP 平台做题时追踪每日进度。

## 功能

- **进度追踪** - 显示今日完成数、目标数、应完成数、进度差
- **智能计算** - 根据工作时间自动计算应完成数量
- **自动计数** - 检测「提交」按钮点击自动 +1，「上一题」自动 -1
- **手动编辑** - 点击完成数可直接修改
- **任务跳转** - 输入任务 ID 快速跳转
- **本地存储** - 数据保存在本地，刷新不丢失

## 安装

1. 下载本仓库
2. 打开 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择文件夹

## 使用

- 底部工具栏显示进度
- 点击「设置」修改每日目标和工作时间
- 输入任务 ID 可快速跳转

## 默认配置

- 每日目标：160 题
- 工作时间：10:00-12:00，13:00-19:00
