# 小羊 AIDP 做题工具

Chrome 浏览器扩展，帮助用户在 AIDP 平台做题时跟踪每日进度。

## 功能特性

- 📊 实时进度追踪 - 显示今日完成数、目标数、应完成数
- ⏰ 工作时间规划 - 自定义工作时间，自动计算进度
- 🔄 自动计数 - 检测提交/上一题按钮，自动更新计数
- 🎨 简洁 UI - 深色主题，底部固定工具栏

## 安装方法

1. 克隆本仓库：
   ```bash
   git clone https://github.com/rnm330/aidptools.git
   ```

2. 打开 Chrome 扩展管理页面：`chrome://extensions/`

3. 开启「开发者模式」

4. 点击「加载已解压的扩展程序」，选择本文件夹

## 使用说明

- 底部工具栏显示当前进度
- 点击数字可直接编辑完成数
- 点击「设置」可修改每日目标和工作时间
- 输入任务 ID 可快速跳转

## 文件说明

| 文件 | 说明 |
|------|------|
| manifest.json | 扩展配置 |
